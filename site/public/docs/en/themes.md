# Themes & Effects

Mox ships **20 built-in themes**, each with its own ambient background effect, rendered live by the Metal GPU.

## Switching themes

Click the appearance button in the top-right, or search a theme name in the command palette. Light/dark themes follow the system appearance.

## Ambient effects

| Effect | Example themes |
| --- | --- |
| Stars | Midnight, Graphite |
| Aurora | Dracula, Lilac Haze |
| Orbs | Liquid Glass, Crimson Night |
| Petals / Maple | Japandi, Red Maple |
| Rain / Snow | Listening Rain, Frost Night |

There's also the living "Animal Island" scene: day/night cycle, drifting clouds, birds, and little critters.

> You can turn effects off in Settings › Effects, or have them follow the system "Reduce Motion" setting.
