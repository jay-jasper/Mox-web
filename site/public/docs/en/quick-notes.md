# Quick Notes

Quick Notes are floating sticky notes you summon instantly — no need to open the editor first. They live in the menu bar, so they're around even after the main window closes.

## Summon

- The global shortcut **⌥⌘N** (customizable in Settings › Shortcuts) creates a note from any app.
- Or use the menu-bar icon › "New Quick Note".

## Note actions

| Action | What it does |
| --- | --- |
| **Pin to screen** | Click 📌 in the title bar — the note stays on top and won't auto-collapse on blur |
| **Dock to edge** | Drag a note to the left/right screen edge; it collapses into a slim drawer — click the arrow to slide it out |
| **Color** | The palette button sets the note's background (Stickies-style) |
| **Save to Desktop** | Save the note as a `.md` file on your Desktop; further edits sync back to it |
| **Close** | The title-bar close button discards the note (a saved file is left untouched) |

## Notes manager

Open it from the menu bar's "Notes" item or the list button on a note:

- Lists both **open notes** and **saved note files** on the Desktop.
- Search at the top; click a row to open/focus; the trash button deletes (open = discard, saved = move to Trash).

## Persistence & restore

- Note state is saved automatically and **restored on quit/relaunch**, including positions.
- Saved notes remember their position by file path and reopen where they were.

> After the main window closes, Mox stays resident in the menu bar so notes are always available; "Quit Mox" from the menu bar exits fully.
