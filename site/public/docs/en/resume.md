# Resume Studio

Write a résumé the structured way: fill sections on the left, see it typeset and paginated live on the right, export ready to send.

## Start

1. File › New Résumé, or import an existing one (File › Import Résumé…).
2. Pick a template.
3. Fill the form on the left; the right pane previews in real time.

## Sections

Common sections: basics, education, experience, projects, skills, certifications, summary, and more.

- **Drag to reorder** sections.
- **Show/hide** the ones you don't need without deleting data.
- Add/remove entries freely within each section.

## Templates & styling

- **Multiple templates** — switch and your content reflows into the new layout.
- Fonts, spacing and accent follow the résumé theme for a consistent look.

## Export

- Export to **PDF** (custom page size and margins) or an image.
- Automatic pagination avoids splitting entries; confirm the layout in the preview before exporting.
