# Shortcuts

## File

| Action | Shortcut |
| --- | --- |
| New | `⌘N` |
| New tab | `⌘T` |
| Save | `⌘S` |
| Close tab / note | `⌘W` |

## Edit

| Action | Shortcut |
| --- | --- |
| Bold | `⌘B` |
| Italic | `⌘I` |
| Find | `⌘F` |
| Command palette | `⇧⌘P` |

## Quick Notes

| Action | Shortcut |
| --- | --- |
| New quick note | `⌥⌘N` |
