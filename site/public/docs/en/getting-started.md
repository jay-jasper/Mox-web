# Getting Started

Welcome to **Mox** — a WYSIWYG Markdown editor built for macOS. Markers slip away, leaving only clean words.

## Install

1. Download the latest build from [GitHub Releases](https://github.com/jay-jasper/Mox-web/releases/latest).
2. Open the `.dmg` and drag Mox into Applications.
3. On first launch, if macOS warns about an unidentified developer, choose "Open Anyway" in System Settings › Privacy & Security.

> Requires macOS 13 (Ventura) or later.

## Workspaces: a folder is all you need

Mox has no proprietary format — it reads and writes your local `.md` files directly, and any plain folder is a workspace.

- Click "Open Folder…" in the sidebar, or press `⌘O`.
- Browse the file tree; right-click a file or folder to create, rename, delete, or reveal in Finder.
- Mount several workspace roots at once and switch freely.
- Drag a file/folder onto the window to open it quickly.

## Tabs & windows

- `⌘T` new tab, `⌘W` close tab, `⌃⇥` cycle tabs.
- `⌘N` new document, `⌘⇧N` new window; windows are independent.
- Closing the main window leaves Mox resident in the menu bar (Quick Notes and the global shortcut stay available); "Quit Mox" from the menu bar exits fully.

## Write your first note

Type and it takes shape:

- `# Heading` becomes an H1; `## / ###` for deeper levels.
- `- ` or `1. ` starts a list; `- [ ]` a task list.
- `> ` for quotes, `---` for a divider, triple backticks for a code block.
- Select text and press `⌘B` bold, `⌘I` italic, `⌘K` link.

Switch between **Rich Text / Source / Split** at the top; `⌘S` to save.

## Make it yours

- "Themes" (top-right) opens the appearance panel — pick any of **20 themes**, the whole window re-skins live.
- "Settings" tunes font, size, line height, accent color, effects toggle, AI endpoint, and more.
