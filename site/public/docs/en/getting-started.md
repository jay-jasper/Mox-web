# Getting Started

Welcome to **Mox** — a Typora-style native macOS WYSIWYG Markdown editor.

## Install

1. Download the latest build from [GitHub Releases](https://github.com/Leejaywell/Mox/releases/latest).
2. Open the `.dmg` and drag Mox into Applications.
3. On first launch, if macOS warns about an unidentified developer, allow it in System Settings › Privacy & Security.

> Requires macOS 13 or later.

## Your first document

- Open a folder as your workspace.
- Create a `.md` file and just start writing — syntax markers show and hide around the caret.
- Use `⌘N` to create, `⌘S` to save.

Next, explore [Themes & Effects](#/en/themes) to set the mood for your writing.
