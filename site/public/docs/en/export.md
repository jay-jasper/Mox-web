# Export

Export your documents to several formats, with the layout visible up front.

## Built-in export

- **PDF**: customize page size, margins, headers/footers and page numbers; confirm the layout in the preview before exporting.
- **HTML**: styled (inlining the current theme) or plain.
- **Image**: export the whole document or a selection as PNG, rendered at 2× for crispness — great for sharing cards.

## Pandoc formats

With [Pandoc](https://pandoc.org) installed, export to Docx, LaTeX, EPUB, reStructuredText and more.

## How to

1. File › Export and choose a target format.
2. Adjust page / style options in the dialog (options vary by format).
3. Pick a destination and you're done.

## Copy instead of export

- Select content and **Copy as Markdown** or **Copy as Rich Text (RTF/HTML)** to paste into email or docs with formatting intact.

## Tips

- PDF export preserves code blocks, math and diagrams exactly as rendered (math is self-contained images, usable offline).
- Export settings are remembered for next time.
