# Knowledge Graph

Weave scattered notes into a network and see how ideas connect.

## Wiki links

- Reference another note anywhere with `[[note name]]`.
- Typing `[[` opens autocomplete — pick an existing note or create one.
- Each note shows its **backlinks** — who references it.

## Open the graph

- Right-click a folder in the sidebar › "Knowledge Graph", or search "Knowledge Graph" in the command palette.
- Build a graph for the whole workspace or a single subfolder.

## Interaction

- **Force-directed layout**: related notes cluster naturally.
- Scroll to zoom, drag to pan, click a node to open its document.
- Node size reflects link count, so hub notes stand out at a glance.
