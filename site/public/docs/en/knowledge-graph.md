# Knowledge Graph

Weave scattered notes into a network and see how ideas connect.

## Wiki links

- Reference another note anywhere with `[[note name]]`.
- Typing `[[` opens autocomplete — pick an existing note or create one.
- Links resolve as you type and are clickable to jump to the target.

## Backlinks

- The bottom of the outline panel lists **backlinks** — "which notes reference this one".
- Following backlinks surfaces the implicit connections between notes.

## Open the graph

- Right-click a folder in the sidebar › "Knowledge Graph", or search "Knowledge Graph" in the command palette (⇧⌘P).
- The graph opens as a **tab**; build it for the whole workspace or a single subfolder.

## Interaction

- **Force-directed layout**: related notes cluster naturally.
- Scroll to zoom, drag to pan, click a node to open its document.
- Node size reflects link count, so hub notes stand out; `#tags` and PDFs appear as distinct node kinds.

## Daily notes

- Press `⌘⇧D` to open/create today's daily note (organized by calendar path) — great with backlinks for journaling and review.
