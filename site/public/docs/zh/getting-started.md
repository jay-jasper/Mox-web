# 快速上手

欢迎使用 **Mox** —— 一款 Typora 风格的原生 macOS 所见即所得 Markdown 编辑器。

## 安装

1. 从 [GitHub Releases](https://github.com/Leejaywell/Mox/releases/latest) 下载最新版本。
2. 打开 `.dmg`，把 Mox 拖入「应用程序」。
3. 首次启动若提示来源未知，在「系统设置 › 隐私与安全性」中允许打开。

> 需要 macOS 13 或更高版本。

## 第一篇文档

- 打开一个文件夹作为工作区。
- 新建 `.md` 文件，直接开始书写 —— 标记会随光标智能显隐。
- 用 `⌘N` 新建、`⌘S` 保存。
