# 快速上手

欢迎使用 **Mox** —— 一款为 macOS 打造的所见即所得 Markdown 编辑器。标记随手隐去，只留下干净的文字。

## 安装

1. 从 [GitHub Releases](https://github.com/jay-jasper/Mox-web/releases/latest) 下载最新版本。
2. 打开 `.dmg`，把 Mox 拖入「应用程序」。
3. 首次启动若提示来源未知，在「系统设置 › 隐私与安全性」中点「仍要打开」。

> 需要 macOS 13 (Ventura) 或更高版本。

## 工作区：文件夹即一切

Mox 没有专有格式 —— 它直接读写你本地的 `.md` 文件，一个普通文件夹就是一个工作区。

- 侧边栏点「打开文件夹…」，或按 `⌘O` 选择目录。
- 左侧文件树可浏览、展开、滚动；右键文件或文件夹可新建、重命名、删除、在 Finder 中显示。
- 可同时挂载多个工作区根目录，在它们之间自由切换。
- 拖一个文件/文件夹到窗口即可快速打开。

## 标签页与窗口

- `⌘T` 新建标签页，`⌘W` 关闭当前标签页，`⌃⇥` 在标签间切换。
- `⌘N` 新建文档，`⌘⇧N` 新建窗口；多窗口各自独立。
- 关闭主窗口后，Mox 仍以菜单栏代理常驻（速记与全局快捷键保持可用）；从菜单栏「退出 Mox」彻底退出。

## 写下第一篇

输入即成形：

- `# 标题` 回车成为一级标题，`## / ###` 为更低层级。
- `- ` 或 `1. ` 开始列表，`- [ ]` 开始任务清单。
- `> ` 引用，`---` 分隔线，三个反引号开始代码块。
- 选中文字 `⌘B` 加粗、`⌘I` 斜体、`⌘K` 加链接。

顶部可在 **富文本 / 源码 / 分屏** 三种模式间切换，`⌘S` 保存。

## 个性化

- 右上角「主题」打开外观面板，**20 套主题**任选，整窗实时换肤。
- 「设置」可调字体、字号、行距、强调色、特效开关、AI 接口等。
