# 快速上手

欢迎使用 **Mox** —— 一款为 macOS 打造的所见即所得 Markdown 编辑器。标记随手隐去，只留下干净的文字。

## 安装

1. 从 [GitHub Releases](https://github.com/Leejaywell/Mox/releases/latest) 下载最新版本。
2. 打开 `.dmg`，把 Mox 拖入「应用程序」。
3. 首次启动若提示来源未知，在「系统设置 › 隐私与安全性」中允许打开。

> 需要 macOS 13 或更高版本。

## 打开工作区

Mox 以「文件夹即工作区」的方式工作 —— 没有专有格式，直接读写本地 `.md` 文件。

- 点侧边栏「打开文件夹…」，或 `⌘O` 选择一个目录。
- 左侧文件树浏览、新建、重命名、删除。
- 支持同时打开多个工作区根目录。

## 写下第一篇

- `⌘N` 新建文档，直接开始书写 —— 输入 `#`、`-`、`>` 等标记会即时变为标题、列表、引用。
- `⌘S` 保存；`⌘T` 新建标签页；`⌘W` 关闭当前标签页。
- 顶部可切换 **富文本 / 源码 / 分屏** 三种编辑模式。
