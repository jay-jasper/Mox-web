# 速记

速记是随手呼出的浮动便签，灵感不必等编辑器打开。它常驻菜单栏，关掉主窗口也还在。

## 呼出

- 全局快捷键 **⌥⌘N**（可在「设置 › 快捷键」自定义），在任意 App 里都能新建一张便签。
- 也可从菜单栏图标 ›「新建速记」呼出。

## 便签操作

| 操作 | 说明 |
| --- | --- |
| **钉在屏幕** | 点标题栏的 📌，便签始终置顶、不随失焦自动收起 |
| **贴边收纳** | 把便签拖到屏幕左/右边缘，自动收成细长抽屉；点边缘小箭头滑出 |
| **换颜色** | 标题栏调色按钮，多种便签底色（类似便利贴） |
| **存到桌面** | 把便签保存为桌面上的 `.md` 文件，之后编辑自动同步回文件 |
| **关闭** | 标题栏关闭键丢弃便签（已存桌面的不影响文件） |

## 便签管理器

从菜单栏「速记列表」或便签上的列表按钮打开：

- 同时列出**打开中的便签**与**桌面已存的速记文件**。
- 顶部可搜索；点一条打开/聚焦；垃圾桶按钮删除（打开中=关闭丢弃，已存=移到废纸篓）。

## 持久与恢复

- 便签状态自动保存，**退出或重启后会恢复**上次打开的便签与位置。
- 已存到桌面的便签按文件路径记住位置，重开回原处。

> 关闭主窗口后，Mox 以菜单栏代理常驻，速记随时可用；从菜单栏「退出 Mox」才彻底退出。
