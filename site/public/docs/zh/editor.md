# 编辑器

Mox 的编辑器是混合式所见即所得：源码与渲染合一，标记只在光标所在处显现，移开即隐去。

## 三种模式

- **富文本**：所见即所得，专注阅读与书写的最终形态。
- **源码**：纯 Markdown 源码，适合精修与粘贴。
- **分屏**：左侧源码、右侧实时预览，双向同步滚动。

顶部工具区一键切换，三种模式共享同一份内容。

## 书写

| 输入 | 效果 |
| --- | --- |
| `# / ## / ###` | 各级标题 |
| `- ` / `1. ` | 无序 / 有序列表 |
| `- [ ]` | 任务清单 |
| `> ` | 引用 |
| `**粗体**` `*斜体*` | 行内强调 |
| 反引号 | 行内代码；三反引号为代码块 |
| `---` | 分隔线 |

## 高效编辑

- **标记折叠**：长文档可折叠标题层级，专注当前章节。
- **智能标点**：中英文标点自动配对与转换。
- **拼写检查**：跟随系统词典。
- **查找替换**：`⌘F` 打开查找栏，支持正则、大小写、整词。
- **图片**：拖入或粘贴图片，按设置存为本地相对路径或上传图床。

## 大纲与状态

- 右上角可呼出**大纲面板**，按标题快速跳转。
- 底部状态栏显示字数、光标行列、阅读时长。
