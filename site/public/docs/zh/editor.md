# 编辑器

Mox 的编辑器是 Typora 风格的混合所见即所得：源码与渲染合一，标记随光标显隐。

## 写作

- 输入 `#` 空格成为标题，`-` 空格成为列表。
- 选中文字按 `⌘B` 加粗、`⌘I` 斜体。
- 行内代码用反引号，代码块用三个反引号。

## 数学与图表

- 行内公式 `$E=mc^2$`，独立公式用 `$$ … $$`（SwiftMath 渲染）。
- 代码块语言写 `mermaid` / `vega` / `graphviz` 即可实时绘图。
- 一键把文档转成 Excalidraw 思维导图画布。

## 知识图谱

用 `[[双链]]` 连接笔记，打开知识图谱查看力导向关系网。
