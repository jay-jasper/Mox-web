# 主题与特效

Mox 内置 **20 套主题**，每套都带专属的动态背景特效（由 Metal GPU 实时渲染）。

## 切换主题

点击右上角的外观按钮，或用命令面板搜索主题名。明暗主题会自动适配系统外观。

## 动态特效

| 特效 | 代表主题 |
| --- | --- |
| 星空 / 星砂 | 子夜、黛 |
| 极光 | 德古拉、紫霭 |
| 流光 | 流光、绯夜 |
| 樱落 / 枫落 | 日栖、丹枫 |
| 细雨 / 霜雪 | 听雨、霜夜 |

还有「动物岛」活场景：昼夜循环、飞鸟流云、草地小动物。

> 在「设置 › 特效」中可关闭动效，或跟随系统「减弱动态效果」。
