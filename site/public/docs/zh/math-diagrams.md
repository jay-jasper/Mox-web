# 数学 · 图表 · 画布

Mox 内置公式、图表与画布渲染，写技术文档与笔记无需离开编辑器。

## 数学公式

由 SwiftMath 原生渲染：

- 行内公式：`$E = mc^2$`
- 独立公式：用 `$$ … $$` 包裹，居中显示。

```
$$
\int_{a}^{b} f(x)\,dx = F(b) - F(a)
$$
```

## 图表

把代码块的语言标注为对应类型即可实时绘图：

- ` ```mermaid ` —— 流程图、时序图、甘特图
- ` ```vega ` / ` ```vega-lite ` —— 数据可视化
- ` ```graphviz ` —— 关系/拓扑图

图表会缓存，编辑时不闪烁。

## 思维导图画布

- 在文档右上角点「画布」，或对 Markdown 执行「生成思维导图画布」。
- 标题层级自动转成 Excalidraw 节点，可自由拖拽、连线、手绘。
- 画布以 `.excalidraw` 保存在工作区，可反复编辑。
