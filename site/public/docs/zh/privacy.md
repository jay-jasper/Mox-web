# 隐私政策

Mox 是一款本地优先的 macOS 编辑器。你的文档保存在你自己的设备上——Mox 不会收集、上传或出售你的内容。

## 我们不收集什么

- 不收集文档内容、文件名或你的编辑行为。
- 不内置分析或遥测（telemetry）。
- 没有账号系统，无需注册或登录。

## 可选的联网功能

部分功能只有在你主动配置并使用时才会联网，且请求直接发往你选择的第三方服务：

- **AI 助手**：调用 AI 功能时，相关文本会发送到你配置的 AI 服务商（如 OpenAI 等）。是否启用、用哪家由你决定。
- **图床上传**：上传图片时，图片会发送到你配置的图床服务。
- **官网与更新日志**：从帮助菜单打开时，会用系统浏览器访问本网站。

这些请求遵循各第三方服务自己的隐私政策；Mox 不经手、不留存这些数据。

## 本地存储

偏好设置、主题、最近文件列表等保存在 macOS 的本地用户目录中，不会离开你的设备。
